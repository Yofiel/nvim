# My NVIM

## Neovim personal configuration for development

### Dependencies

- ![neoclide/coc.nvim](https://github.com/neoclide/coc.nvim) — IDE layer
  - nodejs
  - python
- ![weirongxu/coc-explorer](https://github.com/weirongxu/coc-explorer) — file explorer
- ![nvim-telescope/telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) — a highly extendable fuzzy finder
  - ![sharkdp/bat](https://github.com/sharkdp/bat) (preview)
  - ![sharkdp/fd](https://github.com/sharkdp/fd) (finder)
  - ![BurntSushi/ripgrep](https://github.com/BurntSushi/ripgrep) (finder)
  - ![nvim-treesitter/nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) (finder/preview - already on the config)
  - neovim LSP (already on the config)
- a patched font by ![Nerd Fonts](https://www.nerdfonts.com/#home) for glyphs/icons in terminal.
